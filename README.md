# Quiz App

## 📖 Overview
The **Quiz App** is a web-based application designed to help students test their knowledge across multiple subjects, including:

- Islamiyat  
- Mathematics  
- Computer Science  
- IQ  
- English  
- Pakistan Studies  

Each subject contains **25 multiple-choice questions** with four options, making it an accessible and educational tool for learners.

The app is built with **HTML, CSS, and JavaScript**, using the `quiz_data.js` file as the source for all question data.

---

## 🎯 Purpose
This app provides an **engaging and interactive platform** for students to practice and improve their understanding of key concepts across different subjects.  
The simplified question format ensures it can be used by learners at various levels to **reinforce their knowledge** effectively.

---

## ✨ Features
- **Multiple Topics** → Covers 6 subjects with 25 questions each.  
- **Multiple-Choice Format** → Each question has 4 options with one correct answer.  
- **User-Friendly Interface** → Simple navigation to select topics and answer questions.  
- **Immediate Feedback** → Instantly shows whether the selected answer is correct or incorrect.  
- **Responsive Design** → Works seamlessly on desktops, tablets, and mobile devices.

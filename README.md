**Quiz App** is a web-based educational application designed to help students assess and improve their knowledge across a variety of subjects. The application offers interactive multiple-choice quizzes in the following categories:

* Islamiyat
* Mathematics
* Computer Science
* IQ
* English

Each subject contains **25 carefully selected multiple-choice questions**, with four answer options and one correct answer. The application is built using **HTML, CSS, and JavaScript**, with all quiz data managed through the `quiz_data.js` file.
